按钮点击

攻略

当用户点击页面的某个按钮或其他元素时，我们可以通过js来获取这个事件，并做出响应。
通常需要两步：

  1. 得到某个元素

  2. 监听这个元素的事件

例子： <script>
document.getElementById("button1").onclick = function () {
    alert("我被点击了！");
}
</script> 其中，button1是某个标签的id属性的值。(在任务1中最后一个练习被介绍)
document.getElementById就是通过id来得到元素
onclick对应这个元素的点击事件，当这个事件被触发时，会执行"function() {" 和 "}"之间的代码。
