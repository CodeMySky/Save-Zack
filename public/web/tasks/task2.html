Css是什么，怎么引入

攻略

html相当于创建一个“美女”，那么css就相当于给这个美女“化妆”。
添加css有3种方法：

  1. 给每个标签添加style属性 ←不推荐
    <p style=”color:red”>哈哈</p>

  2. 在head标签里添加<style>标签 ←初学者墙裂推荐 <style>
    p{
        color:red;
    }
    </style>     语法是： 目标 {
        Css样式1;
        Css样式2;
    }

    目标：告诉浏览器，将要对这些目标进行“美化”。通常有3种方式：

    1. 直接用标签←用于更改全部某类标签 p {
          color:red;
      }

    2. 用标签的id属性←用于更改指定标签，方法是“#”+id名： #head {
          color:red;
      }

    3. 用标签的class属性←用于更改含有特定class的标签，方法是“.”+class名 .nav {
          color:red;
      }

  3. 引入css文件 ←nb了一般这么做 <link rel="stylesheet" type="text/css" href="mystyle.css" /> herf属性中填入css文件地址

文字样式

攻略

只要是字都能改，常用的样式有： p {
    color: black;
    font-family: arial, 微软雅黑, 黑体, serif;
    font-size: 20px;
    font-weight: bold;
} color
文字颜色，值可以用red、white等常见英文，或者#fff
font-family
字体，中文值可以用”微软雅黑”、“宋体”、“黑体”，英文可以用arial、”Times New Roman ”(中间有空格要用引号括住字体名称)，不设的话默认使用浏览器默认字体。浏览器会按顺序使用字体，如果第一个字库里没有，就用第二个，如此类推。
font-size
字号，值可以用14px、16px
font-weight
加粗，值可以用bold，不设的话默认为normal

背景

攻略

通常有2个地方需要改背景，一个是按钮、面板等元素的背景，另一个是整个网页的背景
常用的样式有： p {

    background-image: url(/i/eg_bg_02.gif);
    background-positon: 10px 10px
} 整个网页的背景通常给body元素添加样式
